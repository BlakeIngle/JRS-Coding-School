Styling elements

To style an HTML element, you must use CSS.

To style multiple elements, use can use different css selectors.

You can select elements by tag name, class name, by id, and more

CSS selectors

  1. tag name
  2. class name
  3. id

Adding HTML elements

An HTML element can be added to your page using tags.

The p tag represents a paragraph. This is used for paragraphs and other general areas of text on the page. It is the most common tag to use for putting words on a web page.
